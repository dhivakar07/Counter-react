# ⚛️ Counter Application

A simple and interactive **Counter Application** built using **React.js** and **Vite**.  
This project demonstrates the basics of React state management with the `useState` Hook, along with a clean and responsive user interface.

## ✨ Features
- ➕ **Increment Counter**
- ➖ **Decrement Counter**
- 🔄 **Reset Counter**
- ⚡ **Real-time state updates** using React Hooks
- 📱 **Responsive and clean UI**
- ⚛️ **Built with React + Vite**
